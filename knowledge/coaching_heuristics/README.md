# Coaching Heuristics

This directory contains coaching heuristics and decision-making rules for the athlete coaching system.

## Purpose

Coaching heuristics are practical rules of thumb that guide training decisions. Unlike rigid algorithms, heuristics allow for contextual judgment while maintaining consistency in coaching philosophy.

## Structure

- Decision trees for common coaching scenarios
- Workout modification rules based on athlete state
- Recovery and load management guidelines
- Race preparation heuristics
